# Execute e implante seu aplicativo AI Studio

Isso contém tudo que você precisa para executar seu aplicativo localmente.

Veja seu aplicativo no AI Studio: https://ai.studio/apps/ce4dce74-2f5a-489f-b4f6-fd2fd4a2ce9f

## Executar Localmente

**Pré-requisitos:** Node.js


1. Instale as dependências:
   `npm install`
2. Defina a `GEMINI_API_KEY` em [.env.local](.env.local) com sua chave da API Gemini
3. Execute o aplicativo:
   `npm run dev`
